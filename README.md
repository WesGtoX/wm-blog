## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Gatsby](https://www.gatsbyjs.com/docs/tutorial/)
```bash
npm install -g gatsby-cli
gatsby --version
```


### Install and Run

1. Clone the repository:
```bash
git clone https://github.com/WesGtoX/my-blog.git
```
2. Install the dependencies:
```bash
yarn install
```
3. Run:
```bash
yarn start
```
